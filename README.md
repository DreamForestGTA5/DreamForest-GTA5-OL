## 使用说明
### 快捷键
```sh
F5       | 传送至导航点
F6       | 清除通缉
F7       | 满血满防弹衣
F8       | 补满所有武器弹药
F9       | 保存设置并关闭修改器
```

### 注意
- 只有在静止状态下才可以进行传送
- 传送**导航点**时如果传送后卡地形，按下W键即可回到地面
- 传送**导航点**时多人传送需要所有人都下车并按W键
### 为什么传送导航点时会出现这种问题？
- 在进行**导航点**传送时，由于只导航点只有**XY轴**的坐标，并不知道Z轴高度轴的坐标，所以修改器会将你的Z轴改为-225，也就是地下，而GTA5有一个机制，会自动将Z轴位于-225的人物传送至地面。
